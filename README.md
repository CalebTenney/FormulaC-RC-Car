# FormulaC-RC-Car

## Overivew
Fast, light, low-cost RC car built with ESP01.
Read more about it on my [e-portfolio]([(https://jhu.digication.com/caleb-tenney/project-c/published)])

## Features
- Performance oriented design for reaching high speeds
- Servo steering
- Dual battery setup
- Relays driving the motor
- "Slip-clutch" simulation to protect the batteries and motor
  
## Hardware
- ESP01
- Lithium batteries
- Voltage regulator
- Relays
- Servo
- DC Motor
- Legos
- Zip Ties
- Rubber Bands
- Electrical Tape

## Software
- C++
- Arduino IDE
- React Native with Expo
- Expo Go
